TEMARIO

CSS es un leguaje de hojas de estilos creado para controlar aspectos y presentaciones de las paginas web creadas por HTML. En un principio estas paginas estaban hechas por HTML, sin embargo este lenguaje tenia muchas limitaciones cuando se queria hacer el diseño de la pagina y el estilo de sus elementos; esto generaba recargas de codigo con demasiadas etiquetas y atributos.

En base a lo anterior se creo el lenguaje CSS, el cual HTML se ocupa del contenido y CSS se ocupa de los estilos generando dos aspectos en la pagina; Esto conlleva a tener un mayor control en dos leguajes o códigos diferentes y mucho mas claros. Una de las ventajas que contiene el lenguaje CSS es que podemos poner todos los titulos con un mismo aspecto.

Han surgido varias versiones como CSS 1, CSS 2.1 y CSS 3; actualmente se utiliza la versión CSS 2.1 ya que esta no presenta inconvenientes con los navegadores como lo suele ser la versión CSS 3. Para corregir estos problemas se deben crear nuevas versiones de los nevegadores con el soporte adecuado para las nuevas instrucciones.

SINTÁXIS

La sintáxis básica en CSS requiere de un Selector, el cual es un elemento HTML. Seguido del selector se coloca una declaración con una propiedad y un valor determinado. Si se requiere realizar una asignación más completa de propiedades y valores para un elemento HTMl, se debe crear una clase, asginándole un nombre. En el archivo de estilo CSS, dicha clase puede ser usada para establecer propiedades y valores tales como el color de fondo, imagen de fondo, estilo de fuente, entre muchas más. Esto permite modificar con facilidad cada uno de los elementos HTML, desde encabezados, imágenes, botones, etc.
